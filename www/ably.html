About RTKPubcaster Streaming

This ESP32 device is streaming RTK correction data to mobile clients via the Ably realtime platform.

  • Protocol: MQTT over TLS (port 8883)
  • Message Format: JSON envelope with base64-encoded RTCM3 binary data
  • Batching: RTCM3 messages are batched every 200ms for efficiency
  • QoS: Level 1 (at least once delivery) for reliable transmission
  • Conditional Publishing: Only publishes when subscribers are present to save bandwidth
